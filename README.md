## Tic-Tac-Toe (Web Edition)

A clean, modern, and responsive **Tic-Tac-Toe** game built with vanilla JavaScript, HTML5, and CSS3. This project features a sleek dark-mode aesthetic and local score tracking for a two-player experience on the same device.

### üöÄ Features

* **Two-Player Gameplay**: Play locally with a friend on the same screen.
* **Live Scoreboard**: Tracks wins for Player X, Player O, and total draws during the session.
* **Responsive Design**: Optimized for both desktop and mobile screens using CSS media queries.
* **Accessibility**: Includes ARIA roles and keyboard event listeners for improved navigation.
* **Dynamic UI**: Real-time status updates showing whose turn it is or who won the match.

### üïπÔ∏è How to Play

1. **Start**: Player X always starts first.
2. **Move**: Click any empty square on the 3x3 grid to place your mark.
3. **Win**: Align three of your marks horizontally, vertically, or diagonally to win.
4. **Reset**: Use the **Reset** button to clear the current board while keeping the score, or **New Game** to wipe the scores and start fresh.
